repoze.who.plugins.sa releases¶

This document describes the releases of repoze.who.plugins.sa.

Version 1.0.1 (2011-11-23)¶

  • Added ability to prevent from timing attacks. Thanks to Arturo Sevilla!

Version 1.0 Final (2011-05-02)¶

  • Queries issued by this plugin now use a fresh transaction, to avoid using an invalid one (which happens sometimes when repoze.who-friendlyform is used). Thanks to Rick Harding for testing the fix in production before its release!

Version 1.0rc2 (2009-06-27)¶

  • Added repoze.who.plugins.sa.SQLAlchemyUserChecker, a user checker for repoze.who.plugins.auth_tkt.AuthTktCookiePlugin.

Version 1.0rc1 (2009-01-26)¶

  • Introduced the repoze.who.plugins.sa.SQLAlchemyUserMDPlugin metadata provider.
  • Minor docstring fixes.

Version 1.0b3 (2009-01-08)¶

Fixed Bug #56 (User.user_name was not translatable).

Version 1.0b2 (2008-12-18)¶

Renamed repoze.who.plugins.sqlalchemy to repoze.who.plugins.sa due to problems with the namespace.

repoze.who.plugins.sqlalchemy 1.0b1 (2008-12-18)¶

Initial release.
